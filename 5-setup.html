CAMISA

NILES

  • SETUP


  • -> Esperar finalizar a produção do modelo de peça antes de iniciar o setup.
  • -> Pressionar o botão "Reset", para resetar os dois canais.

  • Abrir as portas:
    1. Pressionar o botão "Menu Select".
    2. Pressionar o botão "Manual".
    3. Pressionar o botão "Portas".
    4. Pressionar o botão "Fechar", nas escotilhas dos dois canais.
    5. Girar o botão "Abrir porta", dos dois canais.
  • Fazer a troca da capa da pinça e da flange:
    1. Com ajuda de uma chave allen, retirar os parafusos da tampa que segura a capa da pinça no fuso.
    2. Retirar a tampa do fuso.
    3. Sem a tampa no fuso, retire a capa da pinça.
    4. Com ajuda de uma chave allen, retirar os parafusos da flange do fuso.
    5. Retirar a flange solta do fuso.
    6. Com um pano limpar o fuso.
    7. Colocar a flange do modelo de peças que será produzido.
    8. Colocar e apertar os parafusos na flange com ajuda de uma chave allen.
    9. Colocar a capa da pinça do modelo de peças que será produzido.
    10. Colocar a tampa na capa da pinça.
    11. Colocar os parafusos e apertar, com ajuda de uma chave allen.
    12. Repetir o processo no outro canal.

  • SETUP DOS PROGRAMAS


  • Fazer a troca do programa no PROMOT:
    1. Clicar no botão "Programa escolhido".
    2. Digitar o número do programa que corresponde ao próximo modelo de peça.
    3. Pressionar o botão "Input".
    4. Pressionar o botão "Cycle Start".
    5. Responder as perguntas de acordo com a situação que está as garras do PROMOT, os canais do torno e o MARPOSS.
  • Trocar o programa dos canais do torno Niles:
    1. Pressionar o botão "Menu Select".
    2. Pressionar o botão "Programa".
    3. Ir até a pasta do modelo que será produzido, usando as setas.
    4. Pressionar o botão "Input", para abrir a pasta.
    5. Selecionar o programa que será usado, normalmente os programas que são usados estão com o nome da ferramenta (Bedame) que é usado na usinagem daquele modelo de peças.
    6. Pressionar o botão "Chamada", para chamar o programa.
    7. Pressionar o botão "Channel", para trocar o canal.
    8. Pressionar novamente o botão chamada, para chamar o programa no outro canal.

      9. É importante verificar se foi chamado o programa nos dois canais.


    9. Chamar o programa para o torno pelo "Preparar".
    10. Confirmar troca do programa através do "ativo/ativo".

    Máquina pronta para trabalhar.
