PITTLER

  • LIGAR MÁQUINA

    1. Ligar chave geral.
    2. Soltar o botão de emêrgencia, se estiver precionado.
    3. Fechar as portas dos canais e das entradas para o Jenoptik.
    4. Reset na máquina.
    5. Cancelar alarmes, no botão alarm cancel.
    6. Passar a máquina para JOG. (Para apagar Ref Point)
    7. Selecionar canal um dos canais
    8. Passar a máquina para JOG.
    9. Selecionar o Eixo Z.
    10. Abaixar alguns milimetros nos dois fusos, para fazer o "TEST SI"
  • TEST SI
    1. Pressionar o botão "Menu Select".
    2. Pressionar o botão "Pittler".
    3. Pressionar o botão "Manutenção".
    4. Pressionar o botão "SI TEST".
    5. Pressionar o botão "Iniciar SI TEST".
    6. -> Vai disparar vários alarmes e iniciar a contagem regressiva para o próximo TEST SI. (9 horas)

  • START USINAGEM


  • -> A máquina vazia é o ideal para iniciar o processo.
    1. Fechar o avanço.
    2. Pressionar o botão "Reset".
    3. Pressionar o botão "Auto".
    4. Pressionar o botão "Maskin Start". -> Após pressionar este botão irá iniciar o proceso de usinagem nos dois canais simultaneamente.
  • -> A peça será enviada para o Jenoptik, para ser medida, enquanto isso o processo ficará parado esperando o resultado da medição.

  • Peça OK -> Start da máquina é automatico.

  • Peça NOK -> Máquina ficará parada aguardando correção:
    1. Pressionar o botão "Menu Select".
    2. Pressionar o botão "Pittler".
    3. Pressionar o botão "usinagem".
    4. MC! ou MC2. ( Depende do canal que a peça foi rejeitada)
    5. Pressionar o botão " NOK 100% Correção"
  • -> Inicia usinagem após o botão ser pressionado e será gerado outra peça piloto, com a correção.


  • -> PEÇA PILOTO = é a primeira peça após o inicio do processo, após reset e trocas de ferramentas.

  • -> Após sair a primeira peça aprovada a máquina inicia automaticamente o processo de usinagem.
