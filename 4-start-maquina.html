  • LIGAR MÁQUINA

    1. Ligar chave geral. OK
    2. Soltar o botão de emêrgencia, se estiver pressionado. OK
    3. Fechar as portas dos canais e das entradas para o Jenoptik. OK
    4. Reset na máquina. OK
    5. Cancelar alarmes, no botão alarm cancel. OK
    6. Passar a máquina para JOG. (Para apagar Ref Point) OK
    7. Selecionar canal um dos canais OK
    8. Passar a máquina para JOG. OK
    9. Selecionar o Eixo Z. OK
    10. Abaixar alguns milimetros nos dois fusos, para fazer o "TEST SI" OK
  • TEST SI
    1. Pressionar o botão "Menu Select". OK
    2. Pressionar o botão "Pittler". OK
    3. Pressionar o botão "Manutenção". OK
    4. Pressionar o botão "SI TEST". OK
    5. Pressionar o botão "Iniciar SI TEST". OK
    6. -> Vai disparar vários alarmes e iniciar a contagem regressiva para o próximo TEST SI. (9 horas) OK

  • START USINAGEM


  • -> A máquina vazia é o ideal para iniciar o processo.
    1. Fechar o avanço. OK
    2. Pressionar o botão "Reset". OK
    3. Pressionar o botão "Auto". OK
    4. Pressionar o botão "Maskin Start". -> Após pressionar este botão irá iniciar o proceso de usinagem nos dois canais simultaneamente. OK
  • -> A peça será enviada para o Jenoptik, para ser medida, enquanto isso o processo ficará parado esperando o resultado da medição. OK

  • Peça OK -> Start da máquina é automatico. OK

  • Peça NOK -> Máquina ficará parada aguardando correção: OK
    1. Pressionar o botão "Menu Select". OK
    2. Pressionar o botão "Pittler". OK
    3. Pressionar o botão "usinagem". OK
    4. MC! ou MC2. ( Depende do canal que a peça foi rejeitada) OK
    5. Pressionar o botão " NOK 100% Correção". OK
  • -> Inicia usinagem após o botão ser pressionado e será gerado outra peça piloto, com a correção.


  • -> PEÇA PILOTO = é a primeira peça após o inicio do processo, após reset e trocas de ferramentas.

  • -> Após sair a primeira peça aprovada a máquina inicia automaticamente o processo de usinagem.
